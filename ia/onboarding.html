Job: J1 (activate without anxiety) then opens J-MAIN. Emma (bank), Ravi (private), Claudia (fast).

Type: Flow, Pages, Dialogs, States

Step 1

Onboarding and Activation

The one-time linear chain from "what is this" to the first calm view, by the bank path or the private path. No tab bar the whole way.

Decisions

Recommendations

  • Path Choice gives both paths equal weight, neither preselected (D2).
  • The manual fallback persists on Connect Bank and its error state (dead-end fix).
  • Guided Reveal is one screen, three steps (count, categories, total with action), D1.
  • A person can leave with a partial list and it is saved (dead-end fix).
  • All copy is a specific first draft, marked ? draft copy.
Block order

Nodes 1.1 to 1.5

Page

1.1 Welcome / Value Intro

  1. PHeadline: "See what you are paying for. Calmly."
  2. PSample calm preview before any ask (H1)
  3. PCTA: "Show me how it works"
  4. STrust line: "No bank connection needed to start."
  5. SReturning-user sign in [?]
Page

1.2 Activation Path Choice

  1. PPrompt: "How do you want to start?"
  2. POption A: Connect your bank, read-only
  3. POption B: Add them yourself, 400+ presets (equal weight, D2)
  4. S"Either way, you land on the same calm view."
Page + Dialog

1.3 Connect Bank

  1. PTrust statement: read-only, we can never move money, via Plaid (M2)
  2. P"Choose your bank" → Plaid Link
  3. PPersistent fallback: "Add them yourself instead"
  4. SRegion note: US first [? EU deferred]
Page

1.4 Add Subscription

  1. PPreset search: "Search 400+ services"
  2. PPrefilled, editable form (name, amount, frequency, next date, category)
  3. P"Add subscription" then "Add another"
  4. S"Can't find it? Add it manually."
  5. S"3 added so far. Your list is saved."
Page, 3-step

1.5 Guided Reveal

  1. PStep 1 count: "You are subscribed to 14 things."
  2. PStep 2 categories with logos
  3. PStep 3 total paired with action: "That is $247 a month. Here are 2 you might not use."
  4. PAction: "Review these 2" / "See my list"
  5. SJudgment-free tone throughout

Cross-cutting

SEO, Accessibility, Mobile

SEO. Welcome (1.1) is the only public, indexable node. Everything from 1.2 on is private. See seo.html.

Accessibility. One clear primary action per screen; focus moves to the new H1; the error dialog traps focus, Escape maps to the safe fallback.

Mobile. No structural delta: single-column, full-screen, one action at a time on every breakpoint. Guided Reveal stays gradual at every width.
