Flows and their screens

Flow A - See all recurring charges calmly

J-MAIN · Emma (bank path) · PRIMARY, build first

First session, bank-connected. Welcome to the calm list. This flow sets the pattern the rest clone.

  1. tap 0 Welcome / Value Intro J1 show value before asking for data base
  2. tap 1 Activation Path Choice J1 + J5 two paths, equal weight (D2) base
  3. tap 2 Connect Bank J1 bank path (D5 Plaid US first) base empty error loading
  4. tap 3 Guided Reveal J-MAIN the aha, gradual (D1) base empty
  5. end Home / Subscription List J-MAIN + E1 calm list, count, total · reference screen base save-focus empty error loading

Flow B - Track without sharing bank data

J5 · Ravi (manual + presets)

The privacy path. Shares Welcome, Path Choice, Guided Reveal, and Home with Flow A; the distinct screen is Add Subscription (400+ presets, D2).

  • reuse Welcome / Value Intro shared with Flow A
  • reuse Activation Path Choice shared with Flow A, picks Privacy
  • main Add Subscription J5 privacy path (D2, 400+ presets) base empty error loading
  • reuse Guided Reveal shared with Flow A, empty state matters more here
  • reuse Home / Subscription List shared with Flow A, private list, no bank

Flow C - Find, cancel, and feel the win

J2 + E2 · Claudia

The cut job and the pride moment. Basic cancel is free; the full guide is Pro (D3). Cancel is never walled at the relief moment.

  1. start Home / Subscription List shared with Flow A
  2. tap Subscription Detail J3 decode a charge + J4 its alerts base empty error loading
  3. cancel Cancel Guide J2 identify and cancel; basic free (D3) base empty error
  4. gate Upgrade / Tendd Pro D3 split + D4 price (the gate) · orphan by design base
  5. win Cancel Win Moment E2 the small win, money saved · success-only base
  6. share Share Snapshot S1 privacy-safe share card base error loading

Flow D - Stay ahead of a surprise

J4 · alert to action

Entry is a notification (the return hook). Price change and payment failed are free; trial ending and unusual are Pro (D3).

  1. open Alerts / Activity J4 price change, payment failed base empty error loading
  2. tap Subscription Detail shared with Flow C, "caught it early"
  3. gate Upgrade / Tendd Pro shared with Flow C, a Pro alert type hits the gate
  4. handoff Cancel Guide shared with Flow C, a price change may hand off to cancel
  5. fallback Home / Subscription List shared with Flow A
